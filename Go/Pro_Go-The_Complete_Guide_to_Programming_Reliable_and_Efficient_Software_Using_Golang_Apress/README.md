# Pro Go - Tài liệu học theo chương

Bộ tài liệu này tổng hợp ghi chú học tập cho sách **Pro Go** theo từng chương, theo hướng dễ đọc, dễ dạy lại, và dễ mở rộng cho nhóm học.

## Cách sử dụng
- Mỗi thư mục chapter có một `README.md` theo cùng một cấu trúc.
- Nên điền theo thứ tự: **Tóm tắt nhanh -> Ghi chú chi tiết -> Bài tập -> Tự đánh giá**.
- Với chương mẫu (01, 02), bạn có thể dùng làm chuẩn trình bày cho các chương tiếp theo.
